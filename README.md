# B747-MATLAB-Simulation
Realize the full process simulation based on the Boeing B747 aircraft model in matlab, flying from Chengdu Shuangliu Airport to Beijing Daxing Airport.

You should first run the 'setup.m' file to initialize the project, then run 'Simulation_Start.mlapp' to start the manipulation dashboard, or directly open 'simModel2020.slx' to run the model
